# Anthropic System Prompts Evidence Note

Source: <https://platform.claude.com/docs/en/release-notes/system-prompts>

Retrieved: 2026-05-14

The vendor documentation says Claude web and mobile apps use system prompts to
provide up-to-date context, including the current date, and to encourage certain
behaviors such as Markdown code snippets. The page lists a Claude Opus 4.7
system prompt update dated 2026-04-16 and states that these system prompt
updates do not apply to the Claude API.
